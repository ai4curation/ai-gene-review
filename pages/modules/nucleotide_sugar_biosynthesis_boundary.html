Nucleotide sugar biosynthesis boundary

Boundary module for KEGG ppu00541 in Pseudomonas putida KT2440. The organism-specific map combines several envelope-precursor nucleotide-sugar branches: ADP-heptose for LPS core assembly, Kdo/CMP-Kdo for LPS core linkage, dTDP-rhamnose, GDP-mannose/GDP-D-rhamnose/alginate context, UDP-glucose, UDP-glucuronate, UDP-GlcNAc, and a few unresolved polysaccharide or redox side candidates. It should not be curated as a single linear pathway.
