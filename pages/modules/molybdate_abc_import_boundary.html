Molybdate ABC import boundary

Boundary module for the Pseudomonas putida KT2440 PP_3828-PP_3830 ModABC molybdate ABC importer pulled from the KEGG ppu02010 ABC-transporter umbrella map. ModA is the signal-peptide-bearing periplasmic molybdate-binding component, ModB is the multi-pass inner-membrane permease, and ModC is the ATP-binding energy-coupling subunit with reviewed EC 7.3.2.5 support for ATP-dependent molybdate import. The module records molybdate import as the core function while leaving ModA's tungstate-binding family signal as a secondary substrate lead.
