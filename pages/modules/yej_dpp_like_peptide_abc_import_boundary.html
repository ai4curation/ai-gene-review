Yej/Dpp-like peptide ABC import boundary

Boundary module for Yej/DppA-family peptide ABC import candidates from the KEGG ppu02010 ABC-transporter umbrella map in Pseudomonas putida KT2440. It captures the complete PP_4146-PP_4150 Yej/Dpp-like importer candidate, the partial PP_3076-PP_3078 Yej-like candidate, and the dppA-IV DppA-family singleton.
