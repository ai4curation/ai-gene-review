Methionine import boundary

Boundary module for two MetNIQ-like methionine ABC importer candidates from the KEGG ppu02010 ABC transporter umbrella map in Pseudomonas putida KT2440: PP_0112-PP_0114 (MetQ/MetI/MetN1) and PP_0219-PP_0221 (MetP/MetN2/PP_0221). In each partition, a predicted lipoprotein substrate-binding component is paired with an inner-membrane permease and a reviewed ATP-binding energy-coupling subunit. The available GOA/UniProt evidence supports D-methionine/L-methionine-capable MetNIQ importers, while the physiological substrate range and enantiomer preference in KT2440 remain unresolved.

This boundary module was created from ppu02010 transporter sub-batches, first from the mixed C02 PP_0112-PP_0120 neighborhood and then from the C06 PP_0219-PP_0221 MetNIQ-like neighborhood. It should remain separate from methionine_biosynthesis because it represents import rather than de novo synthesis. UniProt/HAMAP records both L-methionine and D-methionine transport reactions for MetN1 and MetN2, but current GOA supplies D-methionine-specific terms; the in vivo substrate preference in KT2440 remains a knowledge gap.
