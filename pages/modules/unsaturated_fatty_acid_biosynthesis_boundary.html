Unsaturated fatty acid biosynthesis boundary

Boundary/side-node module for KEGG ppu01040 in Pseudomonas putida KT2440. The organism-specific KEGG map contains tesA, tesB, and PP_5331, all reviewed as lipid ester or acyl-thioester hydrolases. The canonical bacterial unsaturated-acyl-ACP branch is covered in the broader fatty_acid_biosynthesis module by FabA/FabB context, not by this ppu01040 thioesterase-only bucket.

Part 1: UFA-route boundary
Unsaturated fatty acid route boundaryMetabolic Pathwayunsaturated_fatty_acid_route_boundary

The ppu01040 membership does not include the FabA/FabB unsaturated-acyl-ACP branch captured in the fatty_acid_biosynthesis module, nor does it contain a desaturase-centered route. This bucket is therefore treated as thioesterase/acyl-pool side context rather than a satisfiable UFA biosynthetic module.
