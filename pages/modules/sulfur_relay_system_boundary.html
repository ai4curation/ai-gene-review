Sulfur relay system boundary

Umbrella boundary module for KEGG ppu04122 in Pseudomonas putida KT2440. The organism-specific sulfur relay map combines molybdopterin/MoCo sulfur-carrier chemistry, Tus/MnmA tRNA wobble-uridine thiolation, ThiS thiamine sulfur carrier context, and rhodanese or mercaptopyruvate sulfurtransferase side nodes. This module indexes those subroutes and should not be used as one generic sulfur-relay pathway for all member genes.

ppu04122 is a KEGG umbrella map for multiple sulfur-carrier systems. Curate and review it through the subroutes below rather than applying one sulfur-relay process to every member.
