Protein turnover peptide-processing boundary

Boundary module for the Pseudomonas putida KT2440 peptide-processing, aminopeptidase, oligopeptidase, and related soluble peptidase split from the protein folding/targeting/turnover functional bucket. The module records enzyme-level peptide-turnover activities and keeps broad proteolysis, hydrolase, and metal-binding annotations as context where more specific functions are available.

First-pass interpretation: this split is a peptide-turnover and soluble peptidase boundary, not a single degradation pathway. Specific aminopeptidase, oligopeptidase, deacetylase, dipeptidase, and metallopeptidase terms were retained where supported. Broad hydrolase, peptidase, metallopeptidase, proteolysis, and metal-binding rows are treated as context when a more informative molecular function exists. PP_2704 illustrates a validation lesson: the UniProt process row GO:0019877 is obsolete in the local ontology, so it was not used as an authored core process.
