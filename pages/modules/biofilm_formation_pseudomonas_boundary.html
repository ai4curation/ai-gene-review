Pseudomonas biofilm-formation boundary

Umbrella boundary module for KEGG ppu02025 biofilm formation in Pseudomonas putida KT2440. The PSEPK primary set is a Pseudomonas aeruginosa-oriented biofilm map that mixes global Gac/Csr/Crp regulation, Wsp-like surface sensing, c-di-GMP turnover, Pil/Chp twitching-motility signaling, type VI secretion context, and orphan regulatory sensors. Curate it through those sub-buckets rather than as one linear biofilm pathway.

This parent module is an index over completed first-pass curation-scale submodules. The submodule reviews carry the gene-level biological claims; this umbrella mainly documents ppu02025 pathway satisfiability and prevents over-curating all members as direct biofilm-formation genes.

Part 1: Umbrella biofilm boundary
Pseudomonas biofilm-formation umbrella boundaryBiological Processbiofilm_formation_umbrella_boundary

ppu02025 is a mixed regulatory and apparatus context for biofilm biology. This parent module indexes the curated sub-buckets and prevents generic biofilm over-annotation across all 43 genes.
