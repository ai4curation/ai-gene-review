Phosphate import boundary

Boundary module for Pseudomonas putida KT2440 high-affinity inorganic phosphate ABC importers encountered during ppu02010 ABC-transporter partitioning. The current curated parts are the PP_2656-PP_2659 PstS/PstC/PstA/PstB1 PstSACB importer and the PP_5326-PP_5329 Pst-like importer. In each system, a PstS-family solute-binding component supplies phosphate binding, two membrane permeases form the transport channel, and a PstB ATP-binding subunit provides ATP-dependent energy coupling. The module also records PP_3818 as a PstS-family phosphate-binding singleton whose cognate permease and ATPase partners remain unresolved.

This boundary module was seeded from C37 and extended with C73 of the ppu02010 partition, then extended with the C51 PP_3818 phosphate-binding singleton. It should remain separate from phosphonate_phosphinate_metabolism because these Pst-like systems import inorganic phosphate, whereas the Phn/Ptx block imports organic phosphonates. Keep open whether KT2440 uses PP_2656-PP_2659 and PP_5326-PP_5329 as two separately regulated phosphate import modules or paralogous variants of one phosphate-limitation transport system.
