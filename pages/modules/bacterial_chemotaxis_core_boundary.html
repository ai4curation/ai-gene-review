Bacterial chemotaxis core Che signaling boundary

Boundary module for the ppu02030 core Che signal-transduction cluster in Pseudomonas putida KT2440. The satisfiable first-pass core covers CheW-like adaptor/scaffold proteins PP_4332 and PP_4333, the CheA histidine kinase, the CheY response-regulator output, CheZ-mediated CheY-P dephosphorylation, and CheB1-mediated methyl-accepting-chemotaxis-protein adaptation. The module deliberately excludes the larger KT2440 receptor panel, orphan MCP/aerotaxis receptors, CheR/CheV accessory adaptation proteins, and periplasmic binding-protein transport spillovers from the broader KEGG ppu02030 map.

First-pass interpretation: this module captures the core PP_4332-PP_4340 Che cluster, not the full bacterial chemotaxis pathway. PP_4332 is the shorter canonical CheW-like protein, PP_4333 is a larger CheW-domain paralog/accessory whose precise partner set is unresolved, CheA/CheY form the main phosphorelay, CheZ resets the output by dephosphorylating CheY-P, and CheB1 tunes receptor sensitivity through methylesterase and glutaminase chemistry.
