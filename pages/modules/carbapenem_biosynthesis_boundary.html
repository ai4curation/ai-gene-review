Carbapenem biosynthesis boundary

Boundary/absence module for KEGG ppu00332 in Pseudomonas putida KT2440. The organism-specific KEGG map contains only ProB and ProA, which are ordinary proline-biosynthesis enzymes that generate glutamate-derived pyrroline-5-carboxylate chemistry. This should not be interpreted as evidence for a complete carbapenem antibiotic biosynthesis pathway in KT2440.
