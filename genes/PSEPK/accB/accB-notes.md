# accB curation notes

- UniProt accession Q88QD5 identifies AccB as the biotin carboxyl carrier
  component of acetyl-CoA carboxylase
  [file:PSEPK/accB/accB-uniprot.txt,
  "This protein is a component of the acetyl coenzyme A"].
- The complete acetyl-CoA carboxylase activity is a property of the assembled
  AccABCD complex. The existing `enables` annotation is therefore marked
  over-annotated, while the core function uses `contributes_to`.
- Complex membership and fatty-acid biosynthetic process are accepted.
