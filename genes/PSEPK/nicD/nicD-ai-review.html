Gene Description

nicD encodes N-formylmaleamate deformylase, an alpha/beta-hydrolase-fold enzyme in the aerobic nicotinate degradation cluster of Pseudomonas putida KT2440. NicD hydrolyzes N-formylmaleamate to maleamate and formate downstream of NicX, completing a key step in nicotinate catabolism.

Proposed New Ontology Terms

N-formylmaleamate deformylase activity

Definition: Catalysis of the reaction N-formylmaleamate + H2O = maleamate + formate + H(+).

Justification: NicD has a characterized substrate-level deformylase reaction in nicotinate catabolism, while the current GO annotation can only use the broader linear-amide hydrolase parent.

Parent term: hydrolase activity, acting on carbon-nitrogen (but not peptide) bonds, in linear amides
