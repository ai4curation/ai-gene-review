# nrfA notes

- UniProt Q6ARF1 names nrfA as ammonia-forming cytochrome c nitrite reductase, EC 1.7.2.2 [file:DESPS/nrfA/nrfA-uniprot.txt "AltName: Full=Ammonia-forming cytochrome c nitrite reductase"; "EC=1.7.2.2"].
- GOA already includes nitrite reductase (cytochrome, ammonia-forming) activity and anaerobic electron transport chain [file:DESPS/nrfA/nrfA-goa.tsv].
- The UniPathway nitrate assimilation annotation is removed because ammonia-forming respiratory nitrite reduction is not the same biological process as assimilatory nitrate use [GO_REF:0000041; file:DESPS/nrfA/nrfA-goa.tsv].
