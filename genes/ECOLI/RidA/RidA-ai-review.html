Gene Description

RidA (formerly YjgF) is a 2-iminobutanoate/2-iminopropanoate deaminase (EC 3.5.99.10) in E. coli that accelerates the release of ammonia from reactive enamine/imine intermediates generated by PLP-dependent enzymes such as threonine dehydratase (IlvA). RidA is a cytoplasmic homotrimer (PMID:10595546) and its primary evolved function is detoxification of reactive intermediates in amino acid metabolism, particularly in the isoleucine biosynthetic pathway. Importantly, RidA acquires chaperone holdase activity ONLY when post-translationally modified by HOCl (hypochlorous acid) via N-chlorination of lysine and arginine residues (PMID:25517874). Untreated RidA has NO chaperone activity. This conditional chaperone function is a secondary, stress-activated moonlighting function and not the core evolved role of RidA.

Core Functions

RidA deaminates reactive enamine/imine intermediates from PLP-dependent enzymes, particularly threonine dehydratase (IlvA), converting 2-iminobutanoate to 2-oxobutanoate and 2-iminopropanoate to pyruvate. This detoxification function supports isoleucine biosynthesis and protects against reactive intermediate toxicity.

  findings:
  - statement: RidA modified with HOCl acts as a highly effective chaperone holdase
  - statement: Untreated RidA has no chaperone activity
  - statement: Chaperone activation requires N-chlorination of lysine and arginine
      residues
  - statement: ridA deletion mutants are sensitive to HOCl stress
  - statement: Chaperone activation is reversible by DTT, ascorbic acid, thioredoxin
      and glutathione

    label: branched-chain amino acid biosynthetic process
  locations:
  - id: GO:0005829
    label: cytosol
  description: RidA deaminates reactive enamine/imine intermediates from PLP-dependent
    enzymes, particularly threonine dehydratase (IlvA), converting 2-iminobutanoate
    to 2-oxobutanoate and 2-iminopropanoate to pyruvate. This detoxification function
    supports isoleucine biosynthesis and protects against reactive intermediate toxicity.
