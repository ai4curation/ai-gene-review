# HSFA1D (AT1G32330) Core Functions Summary

## Current Review Position

HSFA1D is reviewed as an HSE-binding class A heat shock transcription activator that
contributes to Arabidopsis heat-responsive transcription. The current YAML focuses on
the conserved transcription-factor role and HSP90-regulated localization/activity.

## Core Function

The retained core function is RNA polymerase II-specific transcriptional activation
through heat shock element binding, with involvement in cellular response to heat and
positive regulation of transcription. This is supported by conserved HSF domain
biology, UniProt/GO annotations, HSP90 interaction evidence, and Falcon synthesis.

## Non-Retained Draft Claims

Earlier draft summaries proposed distinct new annotations for blue light response,
thermomorphogenesis, circadian regulation, brassinosteroid-linked thermomorphogenesis,
and chilling tolerance. Those claims are not retained in the current YAML because the
available local evidence was not strong enough for HSFA1D-specific new GO
recommendations in this batch.

## Binding Specificity

The generic `protein binding` IPI annotation is recommended for modification to
`GO:0051879 Hsp90 protein binding`, matching the PMID:19704818 and UniProt evidence
for HSFA1D interaction with HSP90-2.
