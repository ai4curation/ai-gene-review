# henmt1 review notes

- Curated in DANRE batch 04. Henmt1 is well supported as the HEN1/piRNA small RNA 2-O-methyltransferase. Oocyte development is real but downstream of piRNA stability, so it is non-core relative to the enzymatic function [PMID:20859253 "Hen1 is required for oocyte development and piRNA stability"].
- Follow-up: broad O-methyltransferase activity is now non-core because small RNA 2'-O-ribose methyltransferase is the core function.
