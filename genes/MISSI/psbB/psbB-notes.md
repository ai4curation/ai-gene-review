## Curation notes

The GOA file for psbB contains only the header row, so there are no existing GOA annotations to review.

Unlike the very small PSII accessory subunits, UniProt describes CP47/PsbB as chlorophyll-binding and helping catalyze primary photochemical processes of PSII. I therefore used the PSII electron-transport process term for the core function, but did not assign oxygen evolving activity because that would overstate CP47's specific contribution.
