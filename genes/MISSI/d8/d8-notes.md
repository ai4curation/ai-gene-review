## Curation notes

- QuickGO export is header-only; therefore no UniProt DR/InterPro-derived terms are listed as `existing_annotations` or proposed as `NEW`.
- Core function uses UniProt's DELLA/GRAS description to represent d8 as a nuclear transcriptional corepressor in gibberellin signaling.
- UniProt flags this sequence as a fragment and cautions that conserved residues needed for feature propagation are missing; the core function is therefore treated as a family-supported DELLA inference that should be confirmed with a full-length Miscanthus sequence. This is also captured as a suggested question in the review.
