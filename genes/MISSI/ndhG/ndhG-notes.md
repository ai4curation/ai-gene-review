# ndhG notes

- The GOA file is header-only, so there are no existing annotations to review.
- UniProt describes NdhG as a chloroplastic NAD(P)H-plastoquinone oxidoreductase subunit that couples quinone reduction to proton translocation.
- The synthesized activity uses `GO:0008137` NADH dehydrogenase (ubiquinone) activity rather than `GO:0050136`, because chloroplast NDH is proton-translocating/electrogenic and the UniProt entry directly cross-references `GO:0008137`.
- The synthesized core function follows the chloroplast NDH role and does not create NEW annotations from UniProt/InterPro predictions.
