# infA notes

- The GOA file is header-only, so there are no existing annotations to review.
- UniProt/HAMAP describes IF-1 as an essential translation initiation component that stabilizes IF-2 and IF-3 binding on the 30S subunit during pre-initiation complex formation.
- The generic cytoplasm location from HAMAP was not used for the core function; the plastid/chloroplast context is supported by the chloroplast gene context and UniProt chloroplast cross-reference.
