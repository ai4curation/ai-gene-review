## Curation notes

The GOA file for psbJ contains only the header row, so there are no existing GOA annotations to review.

UniProt identifies PsbJ as a chloroplast/thylakoid photosystem II reaction-center component. I treated this as a small structural/stabilizing PSII subunit and did not create a `NEW` annotation from the UniProt or HAMAP-derived GO cross-references. I did not assign oxygen evolving activity because UniProt does not support that molecular function for this subunit.
