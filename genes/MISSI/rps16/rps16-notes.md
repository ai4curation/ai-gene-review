## Curation notes

The GOA file for rps16 contains only the header row, so there are no existing GOA annotations to review.

The UniProt record contains chloroplast/plastid localization but also has mitochondrial IEA lines, including a TreeGrafter mitochondrial translation term. Because this is a plastid-encoded chloroplast ribosomal protein, I did not propagate the mitochondrial lines and treated them as likely upstream annotation noise. The core function is supported by the subcellular-location and ribosomal-family statements rather than raw GO cross-reference text.
