# rpl2 notes

- The GOA file is header-only, so there are no existing annotations to review.
- UniProt describes rpl2 as a chloroplastic uL2/50S ribosomal subunit protein with RNA binding and structural constituent of ribosome GO cross-references; the synthesized review uses the more specific rRNA binding term because uL2 is a 50S ribosomal subunit component.
- The rRNA-binding synthesis is supported by the 50S subunit context plus the UniProt feature note for the uL2 RNA-binding domain, rather than relying only on the generic `GO:0003723` InterPro line.
- I did not propagate the TreeGrafter mitochondrial large ribosomal subunit/mitochondrial translation calls or the generic InterPro transferase activity as the synthesized core function.
