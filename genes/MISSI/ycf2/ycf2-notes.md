# ycf2 notes

- The GOA file is header-only, so there are no existing GOA annotations to review or convert into NEW rows.
- UniProt describes Ycf2 as a probable ATPase of unknown function and supports ATP binding and plastid localization [file:MISSI/ycf2/ycf2-uniprot.txt "Probable ATPase of unknown function"] [file:MISSI/ycf2/ycf2-uniprot.txt "GO; GO:0005524; F:ATP binding"] [file:MISSI/ycf2/ycf2-uniprot.txt "SUBCELLULAR LOCATION: Plastid"].
- I used ATP binding rather than ATP hydrolysis activity because the Miscanthus entry directly cross-references ATP binding but not ATP hydrolysis.
- I did not assign a biological process beyond suggested questions because UniProt explicitly says the Ycf2 function is unknown.
