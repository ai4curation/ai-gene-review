# psbT notes

- The GOA file is header-only, so there are no existing annotations to review.
- UniProt describes PsbT as a PSII dimer-interface subunit that plays a role in PSII assembly and dimerization.
- I used `GO:0010207` photosystem II assembly to match the UniProt assembly/dimerization wording, and did not add oxygen evolving activity because the evidence does not support a specific OEC catalytic contribution by PsbT.
