## Curation notes

The GOA file for matK contains only the header row, so there are no existing GOA annotations to review.

UniProt describes MatK as an intron maturase that probably assists splicing of its own and other chloroplast group II introns. I used the specific group II intron splicing term in the core function and used the cached UniProt record only as support, not as a source of `NEW` existing annotations.
