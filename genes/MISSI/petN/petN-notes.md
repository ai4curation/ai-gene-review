# petN notes

- The GOA file is header-only, so there are no existing annotations to review.
- UniProt describes PetN as one of the four small subunits of the cytochrome b6f complex, which mediates electron transfer between PSII and PSI, cyclic electron flow, and state transitions.
- The core function follows the b6f complex electron-transfer role; cytochrome complex assembly remains a related UniProt process but was not made the primary synthesized function.
