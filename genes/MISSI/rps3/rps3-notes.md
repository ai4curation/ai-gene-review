## Curation notes

The GOA file for rps3 contains only the header row, so there are no existing GOA annotations to review.

UniProt identifies rps3 as a chloroplast/plastid uS3/30S ribosomal protein and also contains a TreeGrafter cytosolic small ribosomal subunit line. I treated the TreeGrafter cytosolic line as an upstream annotation mismatch for this plastid gene and did not propagate it.
