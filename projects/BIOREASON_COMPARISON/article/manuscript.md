# BioReason-Pro / AIGR manuscript

The canonical source for the full article is [`manuscript.tex`](manuscript.tex),
with citations maintained in [`references.bib`](references.bib).

Build the PDF from the project directory:

```bash
cd projects/BIOREASON_COMPARISON
just pdf
```

This writes `article/manuscript.pdf` locally. Generated PDFs and LaTeX build
outputs are ignored in this article workspace.
