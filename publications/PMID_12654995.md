A nomenclature is described for restriction endonucleases, DNA 
methyltransferases, homing endonucleases and related genes and gene products. It 
provides explicit categories for the many different Type II enzymes now 
identified and provides a system for naming the putative genes found by sequence 
analysis of microbial genomes.
