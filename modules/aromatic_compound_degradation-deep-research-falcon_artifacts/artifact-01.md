| Protocatechuate branch | Ortho-cleavage of protocatechuate followed by cycloisomerization/decarboxylative steps to the same lower-pathway convergence point | Protocatechuate → 3-carboxy-cis,cis-muconate → 4-carboxymuconolactone/carboxymuconolactone derivatives → β-ketoadipate enol-lactone (or lineage-specific equivalent immediately upstream of β-ketoadipate) | The second core branch of the umbrella, typically initiated by PcaGH and continued by PcaB; downstream details vary among taxa, especially between many Gram-negative bacteria and actinobacteria/rhodococci, so pathway maps should preserve this conditionality rather than force a single universal sequence (pqac-00000010, pqac-00000024, pqac-00000025) |
| Convergent lower pathway | Shared lower funnel from enol-lactone to central carbon metabolism | β-ketoadipate enol-lactone → β-ketoadipate → β-ketoadipyl-CoA → succinyl-CoA + acetyl-CoA → TCA cycle | This is the convergence architecture that defines the umbrella as a catabolic funnel rather than a single linear route; PcaD/PcaIJ/PcaF-like steps couple aromatic carbon to core metabolism (pqac-00000024, pqac-00000025) |
| Meta-cleavage alternative | Extradiol cleavage route for catechol-like intermediates leading through semialdehyde chemistry to central metabolites | Catechol → 2-hydroxymuconic semialdehyde → downstream products such as pyruvate/acetaldehyde | Explicitly excluded from the umbrella’s core definition: meta-cleavage is a neighboring alternative, not part of the ortho-cleavage/β-ketoadipate funnel; it uses extradiol dioxygenases and distinct downstream enzymes (pqac-00000009, pqac-00000010, pqac-00000021, pqac-00000022) |
| Neighboring side-node context | Accessory aromatic decarboxylation and cofactor biosynthesis associated with broad aromatic maps | Aromatic acids ↔ decarboxylated products; FMN → prFMN | UbiD/prFMN chemistry and UbiX flavin prenyltransferase are useful side-node context in broad KEGG-style maps, but they are not obligatory steps of the core ortho-cleavage funnel; they should be shown as adjacent modules rather than merged into the main β-ketoadipate sequence (pqac-00000004, pqac-00000005, pqac-00000006) |
| Excluded neighboring system | Anaerobic aromatic degradation modules that converge by different chemistry | Aromatic substrates → benzoyl-CoA-centered anaerobic intermediates | Explicitly outside the umbrella boundary: anaerobic pathways use different activation logic and central intermediates, so they should not be conflated with the aerobic ortho-cleavage/β-ketoadipate architecture (pqac-00000006, pqac-00000022) |
